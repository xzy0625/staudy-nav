1.1 前端
umi + react + dva + ts

前端使用umi进行开发，主要是看中了umi的页面整合能力，在开发中省了不少力 (umi官网](https://v3.umijs.org/zh-CN)

1.2 后端

1.3 数据库

数据库使用腾讯云数据库进行存储

1.4 服务器

服务器使用腾讯云轻量应用服务器，使用nginx作为服务器进行域名转发

## 快速开始

1.1 安装依赖

```bash
$ yarn
```

1.2 启动本地服务

```bash
$ yarn start
```

1.3 启动umi ui
```bash
$ umi dev
```
